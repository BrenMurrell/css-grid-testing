1. Grid showing displaced primary content based on grid-column and grid-row

This CSS grid uses a grid with columns specified by width: grid-template-columns: 100px 150px auto 150px 100px;. The container element has an 8px padding and we're using the native grid-gap property to specify an 8px gap between grid elements. This layout breaks pretty badly at mobile res.
